Complete Setup & Operations Guide

Welcome to Your School Management System

This guide shows you exactly what to do, in the right order. Start from Phase 0 and go step by step. Click any step to see full details including which menu to open and which button to click.

Step-by-Step Interactive Checklist 97 Detailed Steps Button-Level Instructions

How to use this guide

  1. Use the left menu to jump to any phase (Phase 0, Phase 1, etc.)
  2. Click on any step to expand it and see menu path + buttons
  3. Tick the checkbox when you finish a step - your progress saves automatically
  4. Follow phases in order - later steps need earlier ones (e.g. create Classes before Students)
  5. Scroll to All Modules (A-Z) for a complete screen reference table
  6. Use Search in the left sidebar to find any feature (e.g. "fee", "exam", "library")
