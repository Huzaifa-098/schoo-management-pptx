## How to present (main deck)

1. Open `SchoolSync-System-Presentation.html` in **Chrome** or **Edge** (double-click).
2. Press **F11** for fullscreen presentation mode.
3. Use **→** or **Space** for next slide, **←** for previous.
4. To share as PDF: **Ctrl+P** → Save as PDF → can import into PowerPoint if needed.
